SISTEMA DE EMPRÉSTIMOS

O nosso projeto consiste em um sistema de empréstimos de um colégio, do qual dois tipos diferentes de objetos podem ser emprestados: livros didáticos ou bolas de qualquer esporte.

Para realizar o empréstimo, o usuário precisa ter um cadastro no sistema, do qual seus dados são inseridos pelo administrador. Os dados do usuário são: nome e sobrenome, data de nascimento, CPF, e o ID do aluno do colégio. Com essas informações, o usuário pode escolher um login e senha para realizar os empréstimos, que por sua vez devem ter a data de empréstimo e a de devolução, que será estimada pelo administrador. No ato do empréstimo, deve ser atribuído também o nome do objeto que será emprestado. O administrador pode alterar ou excluir o empréstimo e acrescentar a informação de devolução para quando um objeto for devolvido.

Os objetos são divididos em categorias, que possuem os dados de ID (1 para livro, 2 para bola). Os livros possuem seus próprios atributos, assim como as bolas. O administrador pode cadastrar os objetos, bem como excluir e alterar.

Caso o usuário não devolva os objetos no tempo de devolução estimado, ele receberá uma multa.
